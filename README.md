# SkullGirls-Ranking-System

A lightweight client for a ranked game. It sends match results to a server for processing, allows you to view your rank and stats on a [website](https://skullgirlscommunity.com/danisen), and includes a simple UI to see sent matches and toggle ranked mode on/off.

## Features

- Send match results to the server for rating calculation
- Simple UI to view sent matches
- Toggle Ranked mode on/off

## Website for Rankings

After sending your match results, you can check your rank, stats, and match history on the official website: 

https://skullgirlscommunity.com/danisen

Data is updated after server processing – usually within a few seconds.

Also, keep in mind that Ranked has rules, which you can view on the website. We're not specifically posting the rules here, as they're subject to change.

## Installation

1. Download the latest release from the [Releases](https://github.com/Skullgirls-Communitys-Cut/SkullGirls-Ranking-System/releases) page.
2. Extract the "dbghelp.dll" file to your game folder. "C:\Program Files (x86)\Steam\steamapps\common\Skullgirls" for exemple.
3. That's all!

## Usage
- Use F3 button to show/hide UI
- Enable ranked mode in the UI if you have it disabled.
- Play in the lobby! (Only "All Play" lobby type)
- You can see that your match has been sent to the UI.
